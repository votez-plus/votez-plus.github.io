votez.plus
==========

> Demandez le vote par approbation.

Ceci est la source du site web votez.plus. C'est un site statique généré par
[11ty](https://www.11ty.dev/)


Installation
------------

    $ npm ci

Lancement local
---------------

    $ npm run serve
